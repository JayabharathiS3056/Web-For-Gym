# Web-For-Gym

A modern, responsive gym landing page built using HTML, CSS and JavaScript.

## About the Project

This project is a single-page gym website designed to showcase a fitness center, its classes amenities, reviews and contact details in a clean and attractive layout.

The goal of this landing page is to create a professional online presence for a gym and help attract new members.

## Features

- Responsive landing page layout.
- Attractive hero section with call-to-action buttons.
- About Us section.
- Classes section.
- Amenities section.
- Reviews/Testimonial section.
- Contact section with form.
- Smooth scrolling navigation.
- Mobile-friendly design.

## Sections Included

- Home
- About Us
- Classes
- Amenities
- Reviews
- Contact

## Technologies Used

- HTML5
- CSS3
- JavaScript

## How to Use

1. Download or clone the project files.
2. Open `index.html` in your browser.
3. Explore the gym website sections from the navigation menu.
